# REQ-007: 메인 캘린더 화면

## Status
TODO

## Source
docs/requirements.md §4.1 (메인 캘린더 화면 전체), §1.6.5 (헤더 아이콘), §4.1.7 (인터랙션)

## Summary
앱 진입 화면. 월 단위 7열 그리드에 일기가 있는 날은 무드 아이콘(MoodIcon size=32), 없는 날은 회색 숫자로 표시. 헤더 우측에 검색·통계·리스트 아이콘 3개, 우측 하단에 검정 FAB(펜 아이콘) — FAB 또는 오늘 셀 탭 시 오늘 일기 에디터로 진입.

## User Goal
사용자가 앱을 열자마자 이번 달 감정 흐름을 한눈에 보고, 1탭으로 오늘 일기 쓰기를 시작할 수 있다.

## Acceptance Criteria
- 헤더 우측 3개 아이콘(🔍 검색, 📊 통계, 📋 리스트) — IconButton 컴포넌트(REQ-005) 재사용.
- 월 표시 "M월 ›" 32~40pt, 좌우 스와이프로 월 이동 (또는 ‹ › 화살표).
- 일기 있는 날: 숫자 대신 MoodIcon(REQ-003) 32px. 없는 날: 회색 #C8C8C8 숫자.
- 오늘 날짜 강조: 텍스트 굵게 또는 작은 점 (배경 없음).
- 빈 날짜 탭 → 에디터(REQ-009) 진입 + 즉시 무드 선택 모달(REQ-008) 자동 열림.
- 일기 있는 날짜 탭 → 에디터 진입(기존 데이터 채워짐) + 모달 자동 열림 없음.
- FAB 탭 → 오늘 날짜로 위와 동일 로직.
- 이번 달 외 날짜는 표시 안 함.

## Non-Goals
- 하단 사진 스트립 — v2 (§4.1.6).
- 월/연 picker 모달 — P1 (§8.2).
- 가로 모드 대응 — MVP 제외 (§10.4).

## Dependencies
REQ-002, REQ-003, REQ-005, REQ-006

## Suggested Scope
- 화면 컴포넌트 1개 + CalendarGrid 하위 컴포넌트 1개 + 상단 헤더 조립.
- 100줄 넘어가면 분리 (CLAUDE.md File size 규칙).

## Expected Affected Areas
- Frontend: 캘린더 화면 컴포넌트, CalendarGrid
- Backend: 없음
- Database: 없음
- Tests: 무드 있는/없는 셀 렌더, 월 이동, FAB→에디터 라우팅, 신규 진입 시 모달 자동 호출 플래그
- E2E: 권장 — "앱 열기→오늘 셀 탭→에디터+모달" 골든 패스

## Risk Level
Medium — 진입 화면이라 첫 인상이 여기서 결정됨.

## Suggested Test Coverage
- 일기 1개 있는 달 렌더 시 해당 셀에만 MoodIcon, 나머지는 숫자.
- 좌우 스와이프 시 월 +1/-1.
- 빈 셀 탭 시 라우팅 + 모달 트리거 전달.
