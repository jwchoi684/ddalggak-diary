# REQ-012: 사진 전체화면 뷰어

## Status
TODO

## Source
docs/requirements.md §4.4 (사진 전체화면 뷰어)

## Summary
카로젤에서 사진 탭 시 풀스크린 검정 배경 뷰어. 사진을 화면에 contain으로 가운데 정렬, 좌우 스와이프로 다음/이전 사진, 위/아래 스와이프 또는 X 버튼으로 닫기. 상단에 사진 번호 "3 / 5" 옵션 표시.

## User Goal
사진을 크게 본다. 카로젤 내 다른 사진들도 쉽게 둘러볼 수 있다.

## Acceptance Criteria
- 검정 배경 풀스크린.
- 사진 fit=contain, 가운데 정렬.
- 상단 좌측 ✕ 닫기 + (옵션) 우측 "현재/총합".
- 좌우 스와이프 시 인접 사진으로 전환.
- 위/아래 스와이프 또는 ✕ 탭 시 에디터로 복귀.
- 카로젤에서 탭한 사진이 초기 인덱스로 진입.

## Non-Goals
- 핀치 줌 — v2.
- 사진 공유 — v2.
- 캐러셀 외 위치에서 호출 — 범위 외.

## Dependencies
REQ-011

## Suggested Scope
- Viewer 컴포넌트 1개 + 스와이프 제스처 훅 재사용.

## Expected Affected Areas
- Frontend: PhotoViewer 컴포넌트
- Backend: 없음
- Database: 없음
- Tests: 초기 인덱스, 스와이프 전환, 닫기 시 에디터로 복귀
- E2E: 선택 — "사진 탭→뷰어→스와이프→닫기"

## Risk Level
Low — 표시 전용.

## Suggested Test Coverage
- 초기 인덱스 prop이 그대로 적용.
- 스와이프 좌/우 시 인덱스 +1/-1, 경계에서 멈춤.
- 닫기 시 onClose 호출.
